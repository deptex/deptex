# Product showcase images

Add one image (or video) per slide for the landing page carousel. Use **16:9** aspect ratio.

| Slide | File | Topic |
|-------|------|--------|
| 1 | `slide-1.png` | **Connect** — Connect repo (GitHub/GitLab/Bitbucket), extraction, sync |
| 2 | `slide-2.png` | **Organization** — Teams, roles, integrations, policy-as-code |
| 3 | `slide-3.png` | **Aegis AI** — Chat, automations, tasks, memory, PR review, fixes |
| 4 | `slide-4.png` | **Project Health** — Overview, health score, stats, activity feed |
| 5 | `slide-5.png` | **Dependencies** — Supply chain graph, Watchtower, safe versions |
| 6 | `slide-6.png` | **Security** — Vulns, Depscore, Semgrep, secrets, reachability |
| 7 | `slide-7.png` | **Compliance** — Policy editors, SBOM, licenses, preflight, exceptions |
| 8 | `slide-8.png` | **Notifications** — Custom rules, Slack/Jira/email/PagerDuty, digest |

- Use **PNG or JPG** for images. If an image is missing, the site shows a placeholder.
- For **videos**, add MP4 (or WebM) in `public/videos/showcase/` as `slide-1.mp4` … `slide-8.mp4`. If a video exists for a slide, it is used instead of the image. See `docs/SHOWCASE_VIDEOS.md` for details.
